# Configuration

This folder contains all configuration files:

- `.env` - Environment variables (API keys, secrets) **DO NOT COMMIT**
- `.env.example` - Template for .env file
- `accounts.json` - Z-Library account cookies **DO NOT COMMIT**

## Setup

1. Copy `.env.example` to `.env`
2. Fill in your API keys
3. Add Z-Library cookies to `accounts.json`

See main README for details.
